=========

cctools는 cocos2d-x 게임을 만들 때 유용하게 사용할 수 있는 도구로서 [MIT](./LICENSE) 라이센스를 사용합니다.

## 주요 기능

- cocos2d 캔버스 설정(크기, 해상도 정책 등)
- 위지윅 방식의 cocos2d Node 관리
- 트리 구조로 cocos2d Action 관리 및 시뮬레이팅
- 생성한 Node 및 Action의 코드 변환
- 현재 설정 저장 & 불러오기
